## 遇到的挑战

### 文件格式问题
最初，我发现自己写的文章无法正常显示在博客列表中。经过排查，发现问题出在文件命名和 front matter 上。Jekyll 要求文章文件必须遵循特定格式：

```
YYYY-MM-DD-文章标题.md
```

而且每个文件头部必须包含 front matter：

```
---
layout: post
title: "文章标题"
date: 2025-01-01 12:00:00 +0800
author: "作者名"
---
```

### 响应式设计
为了让博客在手机和平板上也能良好显示，我花了不少时间调整 CSS 媒体查询，确保在不同屏幕尺寸下都有合适的布局。

### 搜索功能
实现全站搜索功能时，我需要考虑如何在不使用后端服务的情况下，实现对所有文章内容的搜索。最终通过 JavaScript 实现了客户端搜索功能。
